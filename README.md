# CaseFun_Auto

          HƯỚNG DẪN CÀI ĐẶT VÀ SỬ DỤNG CASEFUN AUTO V1.1.0

--- BƯỚC 1: CÀI ĐẶT THƯ VIỆN (MỞ CMD/TERMINAL VÀ COPY LỆNH DƯỚI) ---

npm install fs-extra axios ethers https-proxy-agent uuid luxon chalk@4.1.2 node-machine-id readline

--------------------------------------------------------------------

--- BƯỚC 2: CHUẨN BỊ DỮ LIỆU ĐẦU VÀO ---

Tạo các file .txt sau trong cùng thư mục với file p1.js:

1. privatekey.txt    : Mỗi dòng 1 Private Key ví EVM.
2. proxy.txt         : Mỗi dòng 1 Proxy (Định dạng: http://user:pass@ip:port)
3. User_agents.txt   : Mỗi dòng 1 User-Agent (Dùng để giả lập trình duyệt).
4. Config.json       : File cấu hình 

--------------------------------------------------------------------

--- BƯỚC 3: KÍCH HOẠT BẢN QUYỀN (LICENSE) ---

1. Gõ lệnh chạy tool: node p1.js
2. Tool sẽ hiển thị: "=> License Key : "
3. Dán mã Key bạn đã mua vào và nhấn ENTER.
4. Tool sẽ tự động lấy HWID máy của bạn để kích hoạt. 
   (Lưu ý: Key đã nhập sẽ khóa vào máy đó, không dùng được máy khác).

--------------------------------------------------------------------

--- CÁC TÍNH NĂNG TỰ ĐỘNG CỦA TOOL ---

- Auto Login & Sign Message (Xác thực ví).
- Auto Create Username (Nếu tài khoản mới chưa có tên).
- Auto Active Promo Code (Nhận thưởng USDT trải nghiệm).
- Auto Create Case (Tự tạo Case tùy chỉnh theo thuật toán).
- Auto Open Case (Mở 5 lần lấy vật phẩm).
- Auto Upgrade (Nâng cấp vật phẩm vừa mở được để tăng giá trị).
- Auto Claim Tasks (Tự nhận thưởng các nhiệm vụ đã hoàn thành).
